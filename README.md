# pharos-books

Static book catalog for [Pharos](https://github.com/aleksandrr-dev/Pharos),
the free offline classics reader for Android. The app fetches `catalog.json`
and downloads books/covers on demand from this repo's raw URLs.

Every text is public domain (sourced from Project Gutenberg; per-book source
ids are in each JSON's `source` field). Cover art is public-domain / CC0
artwork from Wikimedia Commons. This repo is a distribution artifact —
files are generated by the pipelines in the main Pharos repo (`tools/`),
not edited by hand.
